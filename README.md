# MedBill

**Open-source AI that scans your medical bills, finds errors, and fights back.**

---

Studies suggest a significant percentage of medical bills contain errors — some estimates put it as high as 80%, though the true rate varies. Americans carry $220 billion in medical debt. Insurance companies deny 73 million claims per year — yet fewer than 1% are appealed, even though **more than half are reversed when people do appeal.**

MedBill is free, open-source software that reads your medical bills, flags billing errors, and compares charges against Medicare rates. It runs on your device via [Ollama](https://ollama.com). Nothing is stored. Ever.

> **Disclaimer:** MedBill is an informational tool, not a law firm, medical provider, or licensed professional. Output is generated by automated rules and may contain errors. Do not rely on MedBill output without independent review by a qualified professional.

## How It Works

```
Upload bill photo/PDF
        |
        v
 GLM-OCR (0.9B)      Extracts text, line items, amounts from document
  via Ollama          Runs locally on your device (~4GB RAM)
        |
        v
 Rule Engine          Flags errors using public CMS data:
 (deterministic)      duplicates, unbundling (NCCI), MUE, price outliers
        |
        v
 Results              Flagged issues with estimated overcharge amounts
                      Medicare rate comparison for every line item
```

**No cloud. No API calls. No data leaves your device.**

## What MedBill Does Today

- **Reads your bill** — OCR extraction of line items, descriptions, amounts, dates via GLM-OCR
- **Finds errors** — Duplicate charges, unbundled codes ([NCCI edits](https://www.cms.gov/medicare/coding-billing/national-correct-coding-initiative-ncci-edits)), MUE violations, price outliers vs [Medicare rates](https://www.cms.gov/medicare/payment/physician-fee-schedule)
- **Compares prices** — Every line item benchmarked against national Medicare rates

## Quick Start

**Prerequisites:** Python 3.12+, [uv](https://astral.sh/uv), [Ollama](https://ollama.com)

```bash
# Install Ollama and pull the model (~600MB, one-time)
ollama pull glm-ocr

# Clone and set up
git clone https://github.com/MarcoCornejo/medbill.git
cd medbill
make setup

# Scan a bill
medbill scan your_bill.png

# Or start the web interface
make dev    # http://localhost:8000
```

**Without Ollama:** The app runs in demo mode with sample data so you can explore the UI and rule engine.

**Docker (no setup required):**
```bash
docker compose up --build    # Ollama + GLM-OCR pre-baked
# Open http://localhost:8000
```

## Privacy

MedBill is built on a simple principle: **your medical documents are none of our business.**

- Documents are processed in memory and immediately purged
- No accounts, no logins, no cookies
- No data is transmitted anywhere
- Self-hostable
- Zero cost per document (runs locally)

See [docs/PRIVACY.md](docs/PRIVACY.md) for the full privacy policy.
